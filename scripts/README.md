# Scripts

Utility scripts for the Lore Universe project.

| Script | Purpose | Run once? |
|---|---|---|
| create-structure.js | Creates the initial project folder structure and placeholder files | Yes |
| migrate-obsidian.js | Converts Obsidian `[[double bracket]]` links to `{category\|slug\|display}` syntax; reads from `scripts/staging/{category}/`, writes to `scripts/converted/{category}/` | No — run as needed |
